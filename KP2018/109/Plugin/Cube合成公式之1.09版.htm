Horadric Cube是一个在第二场景中出现的任务道具，它可以使几种不同的任务物品转换成一个过关物品，当然，玩家也可以使用它转化一些普通的物品，甚至可以将它当作储藏箱。Horadric Cube是一个占用物品栏 2*2位置的一个小箱子，但是它提供了 3*4的空间，也就是说带上这个小箱子后，玩家的物品栏多出了8格空间，很不错噢！这是一些喜欢收集装备的玩家又多了一些储藏空间。Horadric Cube的主要作用并不是储藏，而是转换，以下介绍的是各种物品通过Horadric Cube的转换功能所得到的物品。

 

三个同样类型的神符（仅限于低等级的,10#以下）-> 一个高于该等级的神符

三个完美骷髅 + 一件亮黄色杰出装备 + Stone of Jordan（一个暗金色的Unique戒指） -> 为该亮黄色杰出装备打上一个镶嵌孔

一个完美骷髅 + 一件亮黄色杰出装备 + Stone of Jordan（一个暗金色的Unique戒指） -> 转变为更高品质的同类型亮黄色杰出装备

六个完美骷髅 + 一件亮黄色杰出装备 -> 一件随机出现的低品质的同类型亮黄色杰出装备

四个医疗药水（任何类型） + 一个红宝石（任何类型）+ 蓝色魔法物品类/绿色套装类/亮黄色杰出装备类的剑 -> 一把同样类型的可以吸取生命值的剑

三个戒指 -> 一个随机出现项链

三个项链 -> 一个随机出现的戒指

三个医疗药水 + 三个魔力药水 -> 一个恢复药水

三个医疗药水 + 三个魔力药水 + 一个宝石（任何品质）-> 一个全恢复药水

三个恢复药水 -> 一个全恢复药水

三个同类型、同样级别的宝石（级别低于完美型）-> 一个同类型的、高一个等级的宝石

三个最低级宝石 + 一把剑（任何类型）-> 一把同类型的带魔法属性的剑

两袋弩箭 -> 一袋弓箭

两袋弓箭 -> 一袋弩箭

一支长矛 + 一袋弓箭 -> 一堆投掷用标枪

一个斧子 + 两个匕首 -> 飞斧

三个完美的宝石（任何类型）+ 一个魔力物品 -> 一个同类型的新的随机魔法物品

毒气瓶 + 任意类型的医疗药水 -> 一个解毒剂

六个完美宝石（每一个类型一个） + 一个护身符（任意类型）-> 棱镜护身符（提供所有抗性）

一个戒指 + 一个完美绿宝石 + 一瓶解毒剂 -> 一个碧绿戒指（提供抗毒属性）

一个戒指 + 一个完美红宝石 + 一瓶爆炸剂 -> 一个深红戒指（提供抗火属性）

一个戒指 + 一个完美黄玉 -> 一个珊瑚戒指（提供抗电属性）

一个戒指 + 一个完美蓝宝石 + 一瓶解冻剂 -> 一个深蓝戒指（提供抗冰属性）

一面小盾牌（蓝色魔法类或更好的） + 一把钉锤（任意品质） + 两个骷髅 -> 地钉盾

一个钻石 + 一根棍棒（任意类型和品质） + 一把波状刃短剑 + 一根腰带 -> 野蛮的Bardiche

怀特的假腿 + 一本回城魔法书 -> 隐藏关卡：奶牛关
